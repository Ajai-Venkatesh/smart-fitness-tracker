# Smart Fitness Tracker

## Project Description
Smart Fitness Tracker is a mobile application designed to help users monitor their health and fitness activities. The application helps users track their daily calorie intake, protein consumption, and workout activities to maintain a healthy lifestyle.

## Features
- Calorie Tracking
- Daily Protein Intake Monitoring
- Weight Training Exercise Guide
- Cardio Workout Tracking
- Fat Loss Information
- Muscle Gain Guidance
- Weight Gain Tips

## Target Users
- Gym beginners
- Fitness enthusiasts
- People who want to lose or gain weight
- Individuals who want to track daily nutrition

## Tech Stack
Frontend: Flutter  
Backend: Firebase  
Database: Cloud Firestore  
API: Nutrition API  

## Future Improvements
- AI-based diet recommendation
- Personalized workout plans
- Progress tracking graphs
- Smart reminders for workouts and meals

## Project Status
This project is currently under development.
